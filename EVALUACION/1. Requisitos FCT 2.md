# Requisitos Proyecto

## Tipo de proyecto

Creación de una aplicación con frontend y backend, y que haga uso de algún motor de base de datos.

## Duración
Aproximadamente un mes de desarrollo. 

Objetivo tener la aplicación entregada el 30 de Abril de 2025.

## Equipo

- Trabajo en equipo colaborativo
- Metodologías ágiles
- Entregas de valor iterativas

## Requerimientos técnicos
 
- La arquitectura es de libre elección, y deberá ajustarse al tipo de proyecto que seleccionado.
- Tanto en backend como en frontent deben existir test unitarios.
- Despliegues al menos uno o dos entornos
- Los productos deben tener una Base de datos SQL o NOSQL de soporte


## Lenguajes de programación

Como recomendación, proponemos usar .NET (c#) para la creación del backend. 

Para el frontend, recomendamos el uso de React para la componetización de la aplicación. En todo caso el uso de TypeScrit será obligatorio. 

# Otros requerimientos
- Aplicación de principios SOLID en el código que se produzca.
- Buenas estructuras de datos y patrones de diseño

## Control de versiones
- Los proyectos estarán publicados en un repositorio remoto. 
- Cada cambio que se suba al repositorio debe hacerse con un pull request y debe hacerse un code review antes de aceptar los cambios al repositorio.
- Usaremos Conventional Commits (semántica) para los mensajes de confirmación en el repositorio.
- CI/CD: Despliegues con Azure DevOps
- CD. Uno o dos entornos de despliegues a elección del equipo.

## Documentación

El proyecto estará autodocumentado. Para ello se usará un fichero README.MD, en formato markdown, en el que debe existir toda la información necesaria para: 

- Conocer el objetivo principal de la aplicación
- Descripción de las principales funcionalidades
- Estructura del proyecto
- Marco tecnológico
- Despliegue
- Como colaborar
- Licencia de uso

En general cualquier aspecto ya sea funcional o técnico deberá ser recogido en el documento.

# Ideas de aplicación
A continuación se muestran una serie de ideas que pueden servir de inspiración para decidir el proyecto que el equipo quiere realizar. Las aplicaciones son las siguientes: 

- Aplicación de gestión de tareas Permite organizar, priorizar y hacer seguimiento de tareas pendientes con recordatorios y categorización.

- Biblioteca digital de libros: Plataforma para catalogar, gestionar y buscar libros, con opciones de préstamo y recomendaciones.

- Juego de preguntas y respuestas: Aplicación interactiva con cuestionarios temáticos, rankings y modos de juego multijugador.

- Gestor de eventos y reservas: Sistema para planificar eventos, gestionar inscripciones y enviar notificaciones a los asistentes.

- Plataforma de reseñas de cine y series: Un clon de IMDb con funcionalidades para puntuar, comentar y descubrir contenido audiovisual.

- Portafolio digital para artistas: Espacio donde diseñadores, fotógrafos o ilustradores pueden exhibir su trabajo y recibir feedback.

- Sistema de gestión de inventario: Herramienta para controlar existencias, registrar movimientos y generar reportes de stock.

- Plataforma de facturación para autónomos y empresas: Aplicación para crear, gestionar y enviar facturas con seguimiento de pagos.

- Planificador de estudios: App para organizar tareas, exámenes y recordatorios.

- Gestor de notas con Markdown: Un tipo de Notion simplificado con soporte para formateo.

- CRM para pequeñas empresas: Un sistema básico para gestionar clientes y ventas.

- Plataforma de cursos: Tipo Udemy, con lecciones y evaluación.

- App de aprendizaje de idiomas: Flashcards, pruebas y progresión.

- Simulador de entrevistas de trabajo: Preguntas y respuestas con feedback.

- Entrenador de mecanografía: Juego interactivo para mejorar la velocidad de escritura.

- Plataforma de recetas: Con filtros de ingredientes y pasos detallados.

- Registro de ejercicios y progresos: Un diario para rutinas de gimnasio. Integración con Strava.

- Gestor de dietas y calorías: Planificación de comidas con objetivos nutricionales.

- Seguimiento de medicación: Recordatorio para tomar pastillas y control de dosis.

- Controlador de gastos personales: Un gestor financiero sencillo con gráficos.

- Sistema de gestión de alquileres: Para administrar contratos y pagos.

- Generador de portafolio para desarrolladores: Similar a un CV online interactivo.

- Plataforma de foros y debates: Un clon simple de Reddit o Stack Overflow.

- Sistema de reservas para restaurantes/peluquerías: Con calendario y pagos.
